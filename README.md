# Glowworm

<img width="256" height="256" alt="logo" src="https://github.com/user-attachments/assets/ca56c857-869b-475b-a59d-d947dec0f17e" />

Glowworm is an open-source companion app for miHoYo/HoYoverse games taken from an existing project called [Starward](https://github.com/Scighost/Starward) by [Scighost](https://github.com/Scighost).

> *Which is also taken from another existing project called [Collapse Launcher](https://collapselauncher.com) by neon-nyan*

The name is taken from an alterative way of saying Firefly. (If you know what I mean)

It provides the following features:

- Support for Genshin Impact, Honkai: Star Rail and Zenless Zone Zero
- Screenshot Management for Local and Cloud Games
- Support Gacha Records Logging with import and export support for the latest version of UIGF, [UIGF v4.2a](https://uigf.org)
- Support Gacha Logging for Local and Cloud Games
- Support Miliastra Wonderland Gacha Import and Export with UIGF v4.2a

## Installation

Ensure you are using the following:

- Windows 10 from 1809 (17763) up to 22H2 or Windows 11 latest.
- [Microsoft Edge WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH) is installed
- [WebP Image Extension](https://apps.microsoft.com/detail/9pg2dk419drg) is installed
- Animation and Transparency settings as based on your Windows settings, ensure you have it on.
  - Windows 10: Settings (Windows + I) -> Personalization -> Colors
  - Windows 11: Settings -> Accessibility (Windows + U) -> Visual Effects

> Protip: Press the Keyboard Shortcuts to access the section faster. Windows 11, you only need to press the shortcut and you immediately land on the page

> Note: The following listed things should already be installed out of the box, if not you will have to install them.

Visit the [release page](https://github.com/studiobutter/Glowworm/releases) to download and install the app
