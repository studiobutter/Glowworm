New Feature: Miliastra Wonderland Profile Picture Browser

Description: This feature allow user to back up custom profile pictures taken in Genshin Impact - Miliastra Wonderland and browse them in the app. User can then copy, Save to another location or Set as User account's profile picture (with an alert telling if user want to do so).

How it works:
- Global Profile Picture Location: `C:\Users\Khang\AppData\LocalLow\miHoYo\Genshin Impact\webres\BeyondSelfProfile`
- CN Profile Picture Location: `C:\Users\Khang\AppData\LocalLow\miHoYo\原神\webres\BeyondSelfProfile`

The file itself is in png but does not have an extension. File name starts with `NormalSizeBeyondProfile_`. When user visit the page, user will greet with a No image found with a button below it telling to "Get Profile Pictures". When selected, the app will backup the files, add a .png to the end of the file and shows in a grid format like the screenshot area,

Other Info: This is a new Tab ONLY for Genshin Impact, Sidebar Tab icon Glyph: `&#xE77B;`

Goal: Make a image browser that user can browse Miliastra Wonderland Profile pictures taken by the user.
