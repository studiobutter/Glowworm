# Glowworm

<img width="256" height="256" alt="logo" src="https://github.com/user-attachments/assets/ca56c857-869b-475b-a59d-d947dec0f17e" />

Glowworm 是一款开源应用程序，适用于 miHoYo/HoYoverse 游戏，它基于 Scighost 的另一个项目 [Starward](https://github.com/Scighost/Starward) 开发。

*此内容也取自 neon-nyan 的另一个项目 [Collapse Launcher](https://collapselauncher.com)*

名称取自萤火虫的另一种叫法（如果你知道的话）。

它提供以下功能：

- 支持原神、崩坏3和无尽领域零
- 管理PC端和云游戏的截图
- 支持管理和存储抽卡日志，并支持[UIGF v4.2a](https://uigf.org)数据导出
- 支持管理已安装游戏和云游戏的抽卡日志。
- 支持千星奇域抽卡记录导入与导出（基于 UIGF v4.2a 标准）

## 安装

请注意，您必须使用以下系统：

- Windows 10 版本 1809 至 22H2 或最新版本的 Windows 11
- 必须安装 [Microsoft Edge WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH)
- 必须安装 [WebP 图像应用](https://apps.microsoft.com/detail/9pg2dk419drg)
- 动画和透明度设置取决于您的 Windows 设置；请确保已启用这些设置。
  - Windows 10：设置 (Windows + I) -> 个性化 -> 颜色
  - Windows 11：设置 -> 辅助功能 (Windows + U) -> 视觉效果

> 小提示：使用键盘快捷键可以更快地导航到设置。在 Windows 11 中，您只需按下快捷键即可找到所需区域。

> 注意：以下软件已预装在您的计算机上；如果没有，您需要重新安装。

要开发此软件，您需要最新版本的 Visual Studio，并包含以下组件：

- C++ 桌面开发
