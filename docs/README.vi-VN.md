# Glowworm

<img width="256" height="256" alt="logo" src="https://github.com/user-attachments/assets/ca56c857-869b-475b-a59d-d947dec0f17e" />

Glowworm là một Ứng dụng mã nguồn mở cho trò chơi nhà miHoYo/HoYoverse lấy từ một dự án khác gọi là [Starward](https://github.com/Scighost/Starward) bởi [Scighost](https://github.com/Scighost).

> *Cũng được lấy từ một dự án khác gọi [Collapse Launcher](https://collapselauncher.com) bởi neon-nyan*

Tên được lấy từ một cách khác để gọi tên Firefly (Nếu bạn biết)

Nó cung cấp các tính năng sau:
- Hỗ trợ Genshin Impact, Honkai: Star Rail và Zenless Zone Zero
- Quản lý chụp ảnh màn hình cho game trên máy và game trên may (Cloud Games)
- Hỗ trợ quản lý và lưu trữ nhật ký gacha với hỗ trợ xuất dử liệu theo chuẩn UIGF phiên bản 4.2a
- Hỗ trợ quản lý nhật ký Gacha cho trò chơi được cài và trò chơi qua may.
- Hỗ trợ quản ký nhật ký Thiên Tinh Kỳ Vực với hỗ trợ nhập và xuất theo tiêu chuẩn UIGF v4.2a

## Cài đặt

Lưu ý là bạn phải đang dùng những thứ sau:

- Windows 10 từ bản 1809 đến 22H2 hoặc Windows 11 phiên bản mới nhất
- [Microsoft Edge WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH) phải được cài
- [Trình bỗ trợ ảnh WebP](https://apps.microsoft.com/detail/9pg2dk419drg) phải được cài
- Cài đặt hiệu ứng động và độ trong suốt dựa trên cài đặt Windows của bạn, hãy đảm bảo bạn đã bật chúng.
  - Windows 10: Cài đặt (Windows + I) -> Cá nhân hóa -> Màu sắc
  - Windows 11: Cài đặt -> Trợ năng (Windows + U) -> Hiệu ứng hình ảnh

> Protip: Nhấn các phím tắt để di chuyển đên khu vực cài đặt nhan hơn. Trên Windows 11, bạn chỉ cần nhấn phím tắt để tới khu vực cần tìm.
> Note: Những thứ sau đã được cài đặt sẵn trên máy, nếu không thì bạn phải cài lại chúng.

## Dịch thuật

Glowworm sẽ sử dùng Crowdin là dịch vụ dịch thuật chính thức. Chúng tôi hiện đang chuẩn bị chức năng này và sẽ sớm ra mắt. Vui lòng theo dõi để biết thêm chi tiết.

## Phát triển

Để phát triển phần mềm này, bạn sẽ cần cài bản Visual Studio mới nhất với những thứ sau: 

- .NET Desktop Development
- C++ Desktop Development
- Universal Windows Platform Development
