# Glowworm

<img width="256" height="256" alt="logo" src="https://github.com/user-attachments/assets/ca56c857-869b-475b-a59d-d947dec0f17e" />

Glowwormは、miHoYo/HoYoverseゲーム向けのオープンソースのコンパニオンアプリです。[Scighost](https://github.com/Scighost)による既存プロジェクト[Starward](https://github.com/Scighost/Starward)を基に開発されました。


*これは、neon-nyan氏による既存のプロジェクト[Collapse Launcher](https://collapselauncher.com)から着想を得ています*

この名前は、蛍の別の言い方から取られています。（お分かりいただけるでしょうか？）

以下の機能を提供します：

- 原神、崩壊：スターレイル、ゼンレスゾーンゼロに対応
- ローカルおよびクラウドゲームのスクリーンショット管理
- 最新バージョンのUIGF[UIGF v4.2a](https://uigf.org)に対応したガチャ記録のログ機能（インポート/エクスポート対応）
- ローカルおよびクラウドゲームのガチャログ機能
- UIGF v4.2aに対応した星々の幻境ガチャのインポート/エクスポート機能

## インストール

以下の環境を使用していることを確認してください：

- Windows 10（バージョン1809（17763）～22H2）またはWindows 11（最新バージョン）

- [Microsoft Edge WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/?form=MA13LH) がインストールされています。
- [WebP 画像拡張機能](https://apps.microsoft.com/detail/9pg2dk419drg) がインストールされています。
- Windows の設定に基づき、アニメーションと透明度の設定が有効になっていることを確認してください。
  - Windows 10: 設定 (Windows + I) → 個人用設定 → 色
  - Windows 11: 設定 → アクセシビリティ (Windows + U) → 視覚効果

> ヒント: キーボードショートカットを使うと、該当セクションに素早くアクセスできます。Windows 11 では、ショートカットキーを押すだけで該当ページに直接移動できます。

> 注: 以下の項目は、通常インストール済みです。インストールされていない場合は、インストールしてください。


アプリのダウンロードとインストールは、[リリースぺージ](https://github.com/studiobutter/Glowworm/releases)をご覧ください。

## ローカライズ

Glowwormは、デフォルトの翻訳サービスとしてCrowdinを使用します。この機能は現在準備中で、近日中にご利用いただけるようになります。どうぞご期待ください。

## 開発

プロジェクトを直接コンパイルするには、最新バージョンのVisual Studioと以下のワークロードを使用してください。

- .NETデスクトップ開発
- C++デスクトップ開発
- ユニバーサルWindowsプラットフォーム開発
